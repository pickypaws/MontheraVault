Monthera Vault Overview

Introduction

Monthera Vault is a non-custodial staking protocol deployed on the Monad Testnet. Built upon the ERC4626 vault standard, it enables users to earn the native token $MTHR through a transparent, fixed-rate, on-chain reward mechanism.

Its core design ensures that 85% of the token supply is only accessible through staking, promoting real user participation and long-term ecosystem growth.

Token Distribution and Supply Allocation

  • 85% – Locked inside the Monthera Vault reward pool
  • 10% – Allocated for liquidity provisioning: 3% deployed on DEX, 7% held in reserve for future use
  • 4% – Allocated to the core development team
  • 2% – Used as an initial staking seed to bootstrap the vault
  • 1% – Distributed freely via the public claim feature

No further minting will occur, ensuring a fixed, deflationary supply model.

Staking and Reward Mechanism

  1. Users acquire $MTHR via supported DEX liquidity pools.
  2. Stake $MTHR into Monthera Vault.
  3. Begin earning on-chain rewards instantly.

Reward Rate: 1.1902 $MTHR / second, distributed proportionally by vault share.

Vault Token ($sMTHR)

  • Transferable: Users can transfer vault shares.
  • Inheritable Rewards: $sMTHR holders receive rewards automatically.

Key Features

  • Non-custodial staking
  • Fixed-rate rewards
  • Auto-pausing if reward pool is empty
  • ERC4626 compatible

Purpose and Vision

Monthera Vault aims to promote a fair and sustainable distribution of $MTHR, encouraging real user staking and community-driven growth.
